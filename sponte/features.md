# Sponte — 機能一覧

App Store 審査・サポートページ・LP で参照する公式の機能リスト。

実装と一致させること（紹介文と差異が出たら、紹介文ではなくこちらを正とする）。

---

## コア体験 — 1日3つの実行

| 機能 | 概要 |
|------|------|
| モーニングレビュー（画面1） | 起床後、昨日の記録を軽く振り返り、今日の3つを順に書き出す |
| タスクリスト（画面2） | 今日の3つを縦並びで表示。タップで実行画面へ。長押しで並び替え |
| 実行画面（画面3） | 開始／完了の時刻を自動記録。経過時間表示。完了後は画面2へ |
| 余白の時間（画面4） | 3つすべて完了したあとの体験。気分を数値で残せる（任意） |
| 設定（画面5） | プラン状態・1日の区切り時刻・テーマ・タスクテンプレなど |
| 優先順位 | 入力順 = 優先順位（1番目／2番目／3番目）。長押しで並び替え可 |
| ボーナスタスク | 3つでは足りない日のための追加枠。推奨は3つだが、必要なら加えられる |
| ストリーク | 「朝のレビューを完了して3つ決めた日」をカウント。freeze 機能はなし |
| 完了演出 | 全完了時に紙吹雪 + ハプティック |
| 超過ハプティック | 予定時間を超えた瞬間に1回振動 |

---

## 朝のレビュー

| 機能 | 概要 |
|------|------|
| 昨日の振り返り表示 | 前日に取り組んだタスクの完了状況・気分を確認 |
| ストリーク表示 | 朝レビュー継続日数を表示。途切れた場合も静かなトーン |
| 3タスク順次入力 | 1番目／2番目／3番目を順に問いかける形で入力 |
| 0〜2つでも完了可 | 選び切れない朝は1つ、ゼロでも構わない設計 |

問いかけの文面は固定テンプレート（LLM は使用していない）。

---

## 実行（タスクの開始から完了まで）

| 機能 | 概要 |
|------|------|
| 開始ボタン | タップで開始時刻を記録 |
| 自動計測 | 経過時間を画面上に表示 |
| 完了処理 | 完了時に終了時刻を自動記録 |
| 中断・再開 | 必要に応じて中断・再開可 |
| 写真完了フロー | 完了時にカメラ／フォトライブラリから写真を選び、合成して記念に残せる |

## iOS 連携

| 機能 | 概要 |
|------|------|
| Live Activity（ロック画面 + Dynamic Island） | 実行中のタスクをロック画面と Dynamic Island に常時表示 |
| Apple Watch コンパニオン | iPhone を取り出さず、Watch から開始／完了が可能。Watch 単独でも動作 |
| 共有シート連携 | 月次レポート画像・PDF を OS の標準共有シートから書き出し |
| ハプティック | 完了演出・予定時間超過時にフィードバック |

## 振り返り

| 機能 | 概要 | プラン |
|------|------|--------|
| 週ログ | 週単位で完了タスク・気分推移を一覧 | 無料 |
| 週詳細 | 1週間の日別の動きを確認 | 無料 |
| 月ログ | 月単位のカレンダー風ヒートマップ・気分推移・最頻タスク | 無料 |
| 月次レポート（ナラティブ） | その月の記録を自然な日本語の文章で並べ直す。**ルールベース生成（LLM 不使用）**。評価・採点はしない事実ベース文体 | プレミアム |
| 月のテーマ | 月初に1行のテーマを設定。月末レポートの冒頭に反映 | プレミアム |
| 年間ヒートマップ（時間帯別） | 1年分の取り組みを時間帯×日付の格子で表示 | プレミアム |
| PDF 書き出し | 月次レポートを PDF で書き出し | プレミアム |
| 画像書き出し / SNS 共有 | 月次サマリーを画像化して共有 | プレミアム |

## 気分の記録

| 機能 | 概要 |
|------|------|
| 数値入力 | 0〜100 で気分を入力（任意） |
| メモ | 短いコメントを残せる |
| 推移グラフ | 週・月単位で気分の推移を可視化 |

書いても書かなくても良い設計。記入を促す通知はしない。

---

## タスクテンプレ

| 機能 | 概要 |
|------|------|
| テンプレ登録 | よく使うタスク名・推奨時間をテンプレ化 |
| ワンタップ追加 | モーニングレビューでテンプレから素早く追加 |

---

## 活動日の区切り時刻

| 機能 | 概要 |
|------|------|
| 区切り時刻設定 | 「今日」の定義をカレンダー 0:00 ではなく任意の時刻に設定（デフォルト 5:00） |
| 影響範囲 | モーニングレビューの対象日／週ストリップのハイライト／実行画面のタスク取得範囲／完了数カウント |

深夜 2:00 に作業していても、区切りが 5:00 なら前日のタスクとして扱う。

---

## 通知・煽りに関する方針

| 項目 | 対応 |
|------|------|
| プッシュ通知（リマインダー） | 実装していない。ユーザーが意図的に開いたときだけ動くアプリ |
| 競争・ランキング | なし |
| フォロー・SNS 機能 | なし |
| 広告 | なし |
| インタースティシャル等の中断 UI | なし |

---

## プライバシー / データ保存

| 項目 | 対応 |
|------|------|
| データ保存先 | 端末内 SQLite（Drift / `sponte.db`） |
| 自動クラウド同期 | なし |
| 第三者分析ツール | 導入していない（Firebase Analytics / Mixpanel 等は使わない方針） |
| 計測手段 | RevenueCat（課金） / App Store Connect（ダウンロード・クラッシュ）の組み込みのみ |
| 削除ポリシー | soft-delete（`deletedAt` を記録）。物理削除はしない |
| 時刻保存 | UTC 統一。表示時にローカル変換 |

## プレミアム（任意）

決済基盤: **RevenueCat**（`purchases_flutter`）。

### 無料で使える機能

- モーニングレビュー
- 1日3タスクの実行（開始／完了の自動記録）
- ボーナスタスク
- ストリーク
- 気分記録（数値・メモ）
- 週ログ・月ログ（カレンダー表示・気分推移）
- Apple Watch / Live Activity
- タスクテンプレ
- 完了演出・超過ハプティック

### プレミアム限定機能

- 月のテーマ
- 月次レポート（ナラティブ）
- 年間ヒートマップ（時間帯別）
- 月次レポートの PDF 書き出し
- 月次レポートの画像化／SNS 共有

### 解約・購入

- いつでも解約可（App Store / Google Play の標準導線）
- 強制購読導線・解約妨害は実装していない（ダークパターン拒否）
- 復元（Restore Purchases）に対応

## 「やらない」と決めていること

紹介文・サポートページで質問された場合の公式回答用。

- AI / LLM を使った文章生成は **行わない**（月次レポートはルールベース生成）
- 第三者分析ツールは **入れない**
- 広告は **載せない**
- プッシュ通知での煽りは **しない**
- 競争・ランキング・SNS は **作らない**
- ダークパターン（解約妨害・自動更新の不可視化・購入導線の罠）は **採用しない**
