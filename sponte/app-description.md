## アプリ紹介文（本文）

```
やることは分かっているのに、動けない。
タスクを整理しても、計画を立てても、なぜか実行できない。
気がつくと、SNS を眺めて一日が終わっている。

Sponte（スポンテ）は、そんな人のための実行支援アプリです。


■「整理するアプリ」ではなく「始めるアプリ」

多くのタスク管理アプリは、
「やることを整理する」ところまでは得意です。
でも、最初の1つに取りかかるところは、
ずっと自分でやるしかありませんでした。

Sponte は、そこを助けるために作られています。

・「何から始めよう」で迷う時間が減る
・タスク管理アプリを開く回数が減る
・1日の終わりに「今日は何をしたか」を思い出せる

タスク管理アプリではなく、実行管理アプリです。


■ なぜ3つなのか

やることを10個抱えたまま、
「何から始めよう」と考えている時間は、
それ自体がエネルギーを消費します。

Sponte は、その入口を3つに絞ることを薦めます。
（足りない日はあとから加えることもできますが、推奨は3つです。）

数を増やすことが目的ではありません。
3つの中の1つに、最後まで向き合うためのアプリです。


■ 朝、3つだけ決める

毎朝、いくつかの問いに答えるかたちで、
今日の3つを順に書き出します。
1番目・2番目・3番目が、そのまま優先順位になります。

選び切れない朝もあります。
そのときは、1つでも、ゼロでも構いません。


■ 開始すれば、あとは画面に任せる

実行画面で「開始」を押すだけ。
時間の計測も、終了の記録も自動です。
タイマーを止め忘れる心配はありません。

iPhone を取り出さなくても、Apple Watch から開始と完了を操作できます。
ロック画面の Live Activity に、いま取り組んでいる1つが表示されるので、
通知を消したままでも、作業に戻れます。


■ 続けるほど、自分のことが分かる

3つを終えたあとの余白の時間に、
今日の気分を数値で残せます。
書いても、書かなくても構いません。

週・月の単位で、自分の動きを静かに見返せます。
よく動けた曜日、気分の推移、よく実行したタスク。
評価も採点もしません。事実を、自然な日本語で並べ直すだけです。


■ プレミアム（任意）

時間を使ったはずなのに、何も残っていない気がする。
月末にそう感じることがあるなら、Sponte のプレミアムが役立ちます。

毎月の記録を、自然な日本語の短い文章に整理して届けます。
「今月、自分は何に時間を使っていたのか」が、
読み返せる形になります。

・月初に決めた「今月のテーマ」と、月末に届く振り返りレポート
・1年分の取り組みを一枚にまとめた年間ヒートマップ
・月次レポートの PDF / 画像書き出し

無料でも基本機能はそのまま使えます。
いつでも解約できます。


■ プライバシー

・記録は端末内にだけ保存します
・自動でクラウドに送る仕組みはありません
・第三者の分析ツールは入れていません


今日の3つを決める。
いま向き合う1つに集中する。
夜、今日の自分がどう動いたかを、静かに確かめる。

Sponte は、そのための実行支援アプリです。
```

## トーン・方針メモ

紹介文を改稿するときの判断軸。

- **叱らない・責めない・ハードルを上げない** — 全文言で統一
- 「忘れた」「サボった」ではなく「また始められる」
- ストリークが途切れても静かに「前回は X 日。今日からまた」
- 0/3 完了でも「今日また始めよう、それで十分」
- **避ける語彙**: 積み上げる / 成長 / 前進 / 挑戦 / 頑張る / 素晴らしい / いいペース など、自己啓発・励まし寄りの表現
- **AI / 機械学習を訴求しない** — 実装はルールベース（`monthly_narrative_provider.dart`）。LLM を使っていないので、AI を売りにすると虚偽広告になる
- **「3つで止まる」とは書かない** — 機能としてはボーナスタスクで追加可能。「推奨3つ／足りない日はあとから加えられる」と誠実に書く
- 分析ツールは RevenueCat と App Store Connect 組み込みのみ。第三者分析（Firebase / Mixpanel 等）は導入していない方針を明記
- ダークパターン拒否（強制購読導線・解約妨害等は採用しない）
